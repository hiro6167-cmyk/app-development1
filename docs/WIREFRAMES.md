## UI コンポーネント仕様

### ボタン
```
Primary Button:
- 背景: #4CAF50
- テキスト: 白
- 角丸: 12px
- 高さ: 50px

Secondary Button:
- 背景: 透明
- ボーダー: #4CAF50
- テキスト: #4CAF50
- 角丸: 12px
```

### カード（投稿）
```
- 背景: 白
- 角丸: 16px
- シャドウ: 0 2px 8px rgba(0,0,0,0.1)
- パディング: 16px
```

### タブ
```
- 選択時: テキスト #4CAF50, 下線
- 非選択時: テキスト #757575
```

### フォント
```
- タイトル: SF Pro Display Bold, 24px
- 本文: SF Pro Text Regular, 16px
- サブテキスト: SF Pro Text Regular, 14px
- キャプション: SF Pro Text Regular, 12px
```
